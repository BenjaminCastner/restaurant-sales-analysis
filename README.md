# Restaurant Sales Analysis Dashboard

Tools: Excel, Power BI

This project analyses restaurant sales data to identify revenue trends, best-selling items, and peak sales periods.

## Key Analysis
- Revenue trends over time
- Best selling products
- Sales by day of week
- Category performance (burgers, fries, beverages)

## Key Insights
- Burgers generated the highest overall revenue.
- Weekend sales were 35% higher than weekdays.
- Beverages had the highest profit margin.
